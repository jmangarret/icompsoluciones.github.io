Quienes Somos iComp Soluciones.

Somos Desarroladores Web con mas de 10 años de experiencia en el área.

Nuestro fuerte es el Desarrollo de Aplicaciones Web a la medida bajo Frameworks PHP como Laravel, Yii, Symfony y Codeigneiter; empleando librerias AJAX, JQuery, Node, Reactjs y Diseño HTML5, CCS3, Bootstrap y Materialize. Somos amantes de las integraciones de Base de Datos MySQL, PostgreSQL, SQL Server, SAP HANA DB.

Actualmente el avance de la tecnología nos ha obligado a incursionarnos en el mundo de los Webservices, Creación de API's, y Manejo de AWS. Implementación de aplicaciones React JS, Node JS, Express, Nextjs, Deployment (build y bundles.js), Live Reloading, Optimización y Minificación de CSS, SASS, LESS con Webpack, NPM y React Scripts.

Nuestra experiencia con muchas empresas nos ha dado el plus de conocer a fondo el sistema SAP Bussines One haciendo integraciones con otras plataformas y administrando Servicios GNU/Linux como SSH, SFTP, FTP, CRON, SHELL, RDP, CUPS, Apache, Virtual Server y Networking.
